# アプリ名
## Favo Chat
<br>

# 概要
SNSをイメージしたコミニュケーションツールで、主な機能はチャット機能、投稿機能、カレンダー機能の3つです。
チャット機能では、ルームを作成し、他のユーザとチャットコミニュケーションを楽しむことが出来ます。
投稿機能では、自分のマイページから写真とともに文章を投稿し、ブログや日記のような使い方が可能です。
カレンダー機能では、自分だけのスケジュール管理が可能です。
<br>
<br>

# 制作背景（意図）
自分が好きな何かを共有できる人が周りにいない人や、言いづらい人を対象に、それらを共有できる場として、コミニュケーションツールの制作に至りました。
また、代表的なSNSとして挙げられるツールよりも、より手軽にリアルタイムなコミュニケーションを実現できたらと思い、チャット形式で制作しました。
よりユーザ同士が親しくなれるよう、カレンダー機能を実装し、ユーザの予定がわかるようにしました。
<br>
<br>

# DB設計
<br>

## users テーブル

| Column                  | Type   | Options     |
| ----------------------- | ------ | ----------- |
| nickname                | string | null: false |
| email                   | string | null: false |
| encrypted_password      | string | null: false |

### Association

- has_many :rooms
- has_many :messages
- has_many :posts
- has_many :events

<br>
<br>

## rooms テーブル

| Column                  | Type   | Options     |
| ----------------------- | ------ | ----------- |
| name                    | string | null: false |
| info                    | string |             |

### Association

- has_many :users
- has_many :messages

<br>
<br>

## messages テーブル

| Column  | Type       | Options                        |
| ------- | ---------- | ------------------------------ |
| content | string     |                                |
| user    | references | null: false, foreign_key: true |
| room    | references | null: false, foreign_key: true |

### Association

- belongs_to :room
- belongs_to :user

<br>
<br>

## posts テーブル

| Column  | Type       | Options                        |
| ------- | ---------- | ------------------------------ |
| title   | string     |                                |
| text    | text       |                                |
| user    | references | null: false, foreign_key: true |

### Association

- belongs_to :user

<br>
<br>

## events テーブル

| Column      | Type       | Options                        |
| ----------- | ---------- | ------------------------------ |
| title       | string     |                                |
| contents    | text       |                                |
| start_time  | datetime   |                                |
| finish_time | datetime   |                                |
| user        | references | null: false, foreign_key: true |

### Association

- belongs_to :user
